# Task1

This is a Python project for demonstrating Git branching and project scaffolding.

## Git Commands

### Create branches:

```bash
git checkout -b dev
git checkout -b test
git checkout -b master
git checkout -b Husnain



mkdir task1
cd task1
touch README.md
touch requirements.txt
mkdir src
touch src/main.py
mkdir tests
touch tests/test_main.py
```
